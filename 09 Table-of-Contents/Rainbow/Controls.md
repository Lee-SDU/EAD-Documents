## 表单控件

### text

文本输入框。

### password

密码输入框。

### email

电子邮件输入框（HTML5）。

### number

数字输入框（HTML5）。

### url

URL 输入框（HTML5）。

### date

日期输入框。

### datetime

日期时间输入框。

### daterange

日期区间输入框。

### monthly

月度切换。

### btn-group

单选按钮组。

### select

下拉列表框。

### file

文件选择框。

### hidden

隐藏域。

### textarea

多行文本域。

### editor

建议富文本编辑器。

### html

HTML 编辑器。

### chosen

基于 Select 2 的单选控件。

### multiple

基于 Select 2 的多选控件。
