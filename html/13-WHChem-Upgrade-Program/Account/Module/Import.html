万华化学 EAD 升级方案

数据批量导入

在系统初始化或系统出现较多同步异常的时，从 LDAP 获取指定范围的用户账号信息和组织机构信息数据的 Excel 文件导入 EAD 平台对应的数据库表。

数据批量导入顺序

  1. LDAP 批量导出用户账号或组织机构数据 Excel 文件;
  2. 使用数据库工具备份 EAD 平台用户账号和组织机构相关数据库表;
  3. 清除 EAD 平台用户账号和组织机构相关数据库表中带有 LDAP 标识的账号和组织机构数据;
  4. 使用数据库工具导入用户账号和组织机构信息到 EAD 平台数据库;

用户账号信息

EAD 数据表：sys_account
