万华化学 EAD 升级方案

账户体系集成方案

概述

鉴于目前万华化学集团的信息系统都以 LDAP 作为统一的账号和组织数据的管理和推送平台。为了使 EAD 可以满足企业内部统一认证、快速应用开发的目的对 EAD 平台进行基于 LDAP 的账户体系通用集成的升级开发，将 LDAP 中的账号及组织机构数据推送到 EAD 平台。本章节定义本次账户体系通用集成的升级开发的集成方式、数据范围和功能模块。
