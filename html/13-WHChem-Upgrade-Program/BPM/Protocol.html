接口协议

HTTP 方法

所有接口都使用 POST 方法进行请求。

字符集编码

UTF-8

安全认证

HTTP BASIC 认证，BPM 分配应用账号和密码。

请求内容

以 POST 方式通过 Form Data 发送如下表单数据项。

  • action; 固定值 start
  • createTask; 固定值 false
  • params; 传入参数 JSON 字符串。
  • parts; 固定值 all

以上 Form 参数除 params 之外的参数都为 Rest 程序框架固定值的参数。在后续具体接口中定义的请求参数只代表 Form 表单项中 params 参数所对应的 JSON 对象。
