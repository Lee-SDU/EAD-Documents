EAD 平台模型设计

EAD 平台模型设计

Base Model

模板模型，在设计模型时若是特定功能模型，请继承相关模型。

EAD Core Model

EAD 平台核心配置模型，包括视图、模型、数据集、应用等模型设计；

Sys Model

系统管理模块相关设计，包括用户、组织机构、角色、系统动态等模型设计；

Workflow Model

工作流相关模型设计，包括工作流模型、工作流步骤、工作流日志，工作流实例模型设计；
