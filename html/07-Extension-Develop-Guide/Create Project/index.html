扩展开发手册

工程创建

工程创建是指在 Eclipse 中创建引擎工程及扩展开发工程；

引擎工程

  • gbros-root：父级工程，用于依赖jar包及工程间依赖关系管理；
  • gbros-hill：后端引擎核心工程；
  • gbros-web：EAD 平台启动入口工程；

扩展开发工程

扩展开发工程可以有多个工程，按不同的应用，可以建立不同工程。

建立扩展开发工程时，可以导入样例工程，然后进行工程名称和 pmo.xml 文件修改即可。
