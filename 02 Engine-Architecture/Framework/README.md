### 引擎依赖框架

EAD 后端引擎依赖 Mybatis 与 Spring Boot。

#### Mybatis

提供对核心配置库（ead）中配置数据与系统管理库（sys）系统数据的查询能力，即：提供引擎运转所需要的配置数据都是通过 Mybatis 进行获取。

#### Spring Boot

EAD 后端引擎整体依赖于 Spring Boot，包括以下几个方面：

- 引擎启动；
- 核心配置库（ead）中配置数据与系统管理库（sys）系统数据的查询能力；
- 查询扩展驱动的注入、查找；
- 视图动作扩展驱动的注入、查找；
- 变量扩展驱动的注入、查找；
